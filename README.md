除了每天的总结，还有整个项目的收获与反思
收获：对引擎的综合知识得到了扩展，对各种函数和ue特性加深了认识，对简单的小游戏开发过程有一个初步的了解，对ai的使用更加熟练

反思一：缺乏团队合作意识，可以依靠美术组的同学但降低actor的组件复杂度，但却固执的使用默认的坦克与敌人组件，极大增大了actor搭建的难度，浪费了太多时间，甚至只好给EnemyAl写了一套新的逻辑

反思二：不懂得避重就轻，比如护盾特效属于要求之外的，那就先去做其他要求的，这才符合做游戏的工期逻辑，达到效率最大化

反思三：因为不熟悉ue本身的机制，大部分时间花在解决bug上而不是功能实现上，以后要多多学习引擎知识，减少不必要的时间浪费

反思四：要懂得区别使用大模型，逻辑推理严密的使用国外的ai，简单的查找交给国内的足够
