1）创建了四类墙壁，普通，红色激光，蓝色激光，伸缩，通过碰撞预设，实现了不同墙壁的功能，比如对于蓝色激光墙壁，将对Pawn的碰撞预设设置为Ignore，对其他的都是Block，



2）为子弹基类增加了反弹次数的成员变量，为四种不同的子弹和墙壁都也贴上了标签，便于在子弹类中分情况对待不同墙壁遇到不同子弹时是否会反弹并减少反弹次数还是子弹直接穿过去，并且进行判断当子弹击中墙壁并且剩余反弹次数为0时销毁（音波子弹除外）



3）对于伸缩墙，添加了Mover这一actor组件，通过递归的思想创建了AttemptToRise函数：通过定时检测敌人是否在墙所设定的检测盒中，不在则上升，否则进行递归。
