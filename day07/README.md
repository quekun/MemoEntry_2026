1）自己动手，丰衣足食，学习如何制作激光特效

构思上，原本是想着使用现有的物理子弹类，将其速度调至极高来模拟激光，保留轨迹，设定生命周期，后来发现这样容易出现穿墙bug，并且不够真实，彻底放弃物理碰撞引擎，使用射线检测（LineTrace / Hitscan）



行动上，由于没有做过特效，所以大部分问的ai，我输入清晰的指令，对我遇到的困惑表达出我的需求，最后呈现的结果是以ALaserBullet类为基本，辅之以蓝图特效类NS\_LaserBeam



最后的激光由于是视线通道检测，故未产生实体子弹击中特殊墙壁的效果，这个我打算作为一个知识点提醒自己，故不做修改
